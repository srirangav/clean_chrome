README
------

cleanchrome.pl 
By Sriranga Veeraraghavan <ranga@calalum.org>

Cleanchrome is a utility for deleting old versions of Google Chrome onMacOS X.  When 
Chrome is upgraded, at  least  the  previous version appears  to be retained.   This 
utility can be used to delete such older versions of Chrome.

cleanchrome is similar to OldChromeRemover for Windows  by  Shane  Gowland:

http://shanegowland.com/software/2012/oldchromeremover-module/
